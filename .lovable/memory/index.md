# Project Memory

## Core
RankMyMatch — padel ranking app. PT-BR. Mobile-first.
Design: "Carbon Fiber Precision" — dark carbon bg, rally neon green accent, Inter+Space Grotesk.
Lovable Cloud (Supabase) for backend. Google OAuth only.
27+ tables with RLS. Elo rating system adapted for padel.
Bottom nav: floating rounded pill with 5 tabs.
Cards: rounded-3xl, border border-border, bg-card.
